## 🎮 遊戲簡介

**暗影協定 Shadow Protocol** 是一款結合**潛行策略 × 資訊安全 × AI 教育**的同步回合制策略遊戲。

玩家操控數位戰場上的智能程式，在光明與陰影之間進行攻防。忍者的潛行、守衛的掃描、陷阱的佈置——所有這些在這裡都是真實的資安攻防演算法模擬。

這不只是一款遊戲，而是一個讓學生自然地理解**加密、入侵偵測、AI 策略**的互動學習環境。

---

## ⚙️ 核心玩法

### 同步回合制（Simultaneous Turns）
玩家與 AI 同時提交行動，同時結算。消除傳統回合制的等待感，製造真正的策略張力。

### 光影系統（Shadow & Light）
- **陰影格** = 加密通道：程式在其中完全隱身
- **光照格** = 開放通道：暴露在掃描之下
- 熄燭、投石、設陷——每個動作都影響戰場的光影格局

### 聽覺系統（Sound Detection）
聲音穿透牆壁傳播，不同動作產生不同聲源半徑。走路、奔跑、武器、倒地——每個聲音都可能洩露你的位置，也可以被用來製造誘餌。

### 三種控制模式
| 模式 | 說明 | 提示符 |
|------|------|--------|
| **直控 Direct** | 按鍵即行動，適合新手體驗 | `:` |
| **指令 Command** | 一次規劃 N 步，提交後執行 | `>` |
| **腳本 Script** | 以 Blockly 積木撰寫 AI 自動行為 | `$` |

## 🎓 教育應用

Shadow Protocol 的每個遊戲機制都對應真實的資安概念，讓學習發生在「玩」的過程中：

| 遊戲體驗 | 對應學習內容 |
|----------|-------------|
| 在陰影中隱身 | 加密與隱私保護 |
| 守衛掃描光錐 | 入侵偵測系統（IDS）原理 |
| 聲音吸引敵人 | 封包流量分析 |
| 用 Blockly 寫腳本 | 程式邏輯與演算法設計 |
| RL Agent 對戰 | 強化學習基礎概念 |
