
# Shadow Protocol｜控制模式與可見性規格（Final Draft）

> 目的：把 **控制模式** 與 **可見性** 收斂為一套簡潔、一致、可實作的規範，並規劃章節預設搭配。

---

## 1) 控制模式定義

### 直控（Direct）
- **操作**：玩家按鍵即行動，逐步即時執行（等同 N=1）。
- **用途**：新手體驗規則與手感、快速試探。

### 指令（Command）
- **操作**：一次提交 **恰好 N 步**（建議章節用 N=6 / 8）。
- **流程**：規劃 →（可預視風險）→ 提交 → 逐步執行 → 回饋。
- **建議語彙**：`UP/DOWN/LEFT/RIGHT, A/B/X/Y（技能）, WAIT, TURN L/R`。

### 腳本（Script）
- **操作**：完全依 **Blockly 腳本** 自動運作。
- **資料來源**：AI/積木 **只使用可感知到的資訊**（不作弊）。

> UI 建議：終端提示符 `直控: >`、`指令: $`、`腳本: §`。

---

## 2) 可見性定義

### 開燈（Bright）
- **顯示**：整張地圖 + 所有單位（敵/我）+ 所有亮區與光源。
- **暗幕層**：關閉。

### 迷霧（Fog）
- **顯示**：
  - 地形 / 物件（牆、門、道具、**蠟燭等光源**）
  - **我方單位**：永遠可見（含位置/朝向/狀態）
  - **敵方單位**：滿足下列任一條件才顯示：  
    `EnemyVisible = (Lit && InTeamFOV) || InTeamNV`
    - **Lit**：處於亮區（被光照）
    - **InTeamFOV**：在任一我方單位的視域內
    - **InTeamNV**：在任一我方單位的夜視半徑內
- **暗幕層**：半透明（例：opacity 0.5）。
- **備註**：FOV/NV 為隊伍聯集（任何一人看到＝全隊 UI 顯示）。

### 關燈（Dark）
- **顯示**：僅 **自身 FOV** 內的地形/物件/單位與亮區（FOV 外一律不可見）。
- **暗幕層**：全黑（以 FOV 打洞）。
- **回放**：通關後可切換「開燈」觀看回放。

#### FOV / NV 建議
- **FOV**：受牆體遮擋，隨朝向更新。
- **NV（夜視）**：建議採 **受牆遮擋** 的近距離視域加成（更直覺、避免穿牆透視）。

#### 光源/蠟燭（預警機制）
- 亮區固定顯示其光暈。
- **熄滅事件**：忍者可熄燭 → 產生短噪音 + 光變化事件。
- UI/終端範例：`[光源] 北廊燭火熄滅（疑似入侵）`。  
  → 作為守衛的巡查/封鎖優先區。

## 3) 章節搭配（控制 × 可見性，預設值）

| 章節 | 角色 | 直控 | 指令 | 腳本 |
|---|---|---|---|---|
| 第一章 | 忍者 | **開燈** | **迷霧** | **關燈** |
| 第二章 | 守衛 | **迷霧** | **迷霧** | **關燈** |

> 原則：**腳本一律關燈**（AI/積木只能取得 FOV 內資訊）；守衛章避免一次暴衝難度，直控與指令都維持「迷霧」。

---

## 4) 可視化規則一覽（實作備忘）

- **Konva 暗幕層**：
  - 開燈：隱藏
  - 迷霧：`opacity = 0.5`
  - 關燈：`opacity = 1` + 以當前角色 FOV 打洞
- **單位可視**：
  - 開燈：敵/我 **全顯**
  - 迷霧：我方 **全顯**；敵方依 `EnemyVisible` 判定
  - 關燈：僅 FOV 內顯示
- **淡出**：失去視覺條件時，敵方 0.2s 淡出，避免突兀。
- **終端事件**（例）：`[可視] 目標短暫曝光（Lit∧FOV）`、`[夜視] 近距離偵測`、`[可視] 目標脫離視域（最後已知 x,y）`

---

## 5) 指令模式的小規則（簡）

- **固定步數 N**：每回合必須恰好 N 步（不足以 `WAIT` 補齊）。
- **規劃預視**：提交前標示每步的風險（將入視錐/產噪/撞牆）。
- **中斷**：被偵測/致命事件發生即在當步停下，終端標示原因與步序。

---

## 6) 設計取向總結

- 只保留三種**控制模式**：直控 / 指令 / 腳本。  
- **可見性**分三級：開燈 / 迷霧 / 關燈；以章節預設搭配呈現節奏。  
- 忍者線：開燈 → 迷霧 → 關燈；守衛線：迷霧（直控/指令）→ 關燈（腳本）。  
- 迷霧的關鍵：**我方可見**、**敵方需 Lit∧FOV 或 NV**；蠟燭熄滅成為**預警訊號**。
