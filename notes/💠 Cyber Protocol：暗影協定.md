
# 💠 Shadow Protocol：暗影協定  
## 四大系統模組職業設定總覽  

---

## 🥷 滲透模組（Infiltrator / 忍者程式）

**定位**：滲透與破壞。潛行能力強，專精欺敵與暗殺，代表惡意程式或Exploit Script。  
**世界觀象徵**：黑帽駭客的攻擊代碼，在封閉網路中搜尋漏洞、奪取資料。  
**主要功能**：暗影潛行、假聲欺敵、快速刺殺、能量靜止回復。  
**核心特色**：高機動、低噪音、能量效率佳；但血量低、無法長期交戰。  
**戰術應用**：適合單點突破、滲透高價值區域、擾亂敵方感知網。  

| 屬性 | 數值（0~10） | 說明 |
|------|----------------|--------|
| ❤️ 生命 | 4 | 脆弱但可閃避。 |
| 🔋 能量 | 6 | 靜止時自動回復。 |
| 👁 視距 | 5（夜視45°） | 不需光源即可觀察。 |
| 👂 聽距 | 3 | 一般水準，易被偷襲。 |
| 🦶 移速 | 2 | 靜步移動以避免聲響。 |
| 🔊 聲音量 | 1 | 幾乎無聲。 |
| ⚔️ 攻擊力 | 5 | 高爆發近戰刺殺。 |

---

## 👮 防毒模組（Antivirus / 警衛程式）

**定位**：系統防禦核心。以光照（掃描）維持控制區域，需休眠回能。  
**世界觀象徵**：防毒引擎與監控程序（Security Daemon）。  
**主要功能**：掃描照明、區域防禦、假睡誘敵、喚醒其他模組。  
**核心特色**：穩定但笨重，能量高但需周期性休眠。  
**戰術應用**：維持防線、配合警犬巡邏、掩護工人修復節點。  

| 屬性 | 數值（0~10） | 說明 |
|------|----------------|--------|
| ❤️ 生命 | 6 | 穩定耐久。 |
| 🔋 能量 | 8 | 高能但依賴休眠補充。 |
| 👁 視距 | 3（90°光錐） | 光源主導，黑暗中視野差。 |
| 👂 聽距 | 3（睡眠時=1） | 一般偵測力。 |
| 🦶 移速 | 1 | 反應慢。 |
| 🔊 聲音量 | 2 | 行動有金屬摩擦音。 |
| ⚔️ 攻擊力 | 4 | 中距穩定打擊。 |

## 👷 工作模組（Worker / 維運程式）

**定位**：系統維運與結構改造，唯一能操作地形與資源節點的單位。  
**世界觀象徵**：背景服務程序（System Service）。  
**主要功能**：挖礦產資、破牆開路、修復燈塔、推動方塊重組地形。  
**核心特色**：高耐久與無限能量供應，但幾乎沒有感知與攻擊力。  
**戰術應用**：維持伺服器穩定、開啟新路、支援防禦模組運作。  

| 屬性 | 數值（0~10） | 說明 |
|------|----------------|--------|
| ❤️ 生命 | 8 | 高耐久。 |
| 🔋 能量 | 10（自動回復） | 永不耗盡。 |
| 👁 視距 | 2 | 極短，視覺弱。 |
| 👂 聽距 | 1 | 幾乎無聽覺。 |
| 🦶 移速 | 1 | 行動緩慢。 |
| 🔊 聲音量 | 4 | 操作噪音大，容易暴露。 |
| ⚔️ 攻擊力 | 0 | 無攻擊能力。 |

---

## 🐕 偵測模組（Sensor / 警犬程式）

**定位**：高靈敏移動感測單位，負責廣域偵測與聲音警報。  
**世界觀象徵**：主動監控 AI、行為偵測演算法（Behavioral Detection Engine）。  
**主要功能**：夜視感測、反光標記、咬合控制、吠叫警報、休眠回能。  
**核心特色**：高速巡邏與夜間偵測能力，但生命低、容易暴露。  
**戰術應用**：巡邏前線、同步守衛光源、第一線感知異常訊號。  

| 屬性 | 數值（0~10） | 說明 |
|------|----------------|--------|
| ❤️ 生命 | 3 | 脆弱。 |
| 🔋 能量 | 7（休眠回復） | 行動穩定但需休息。 |
| 👁 視距 | 5（夜視） | 黑暗中活動自如。 |
| 👂 聽距 | 5（睡眠時=2） | 優異感測力。 |
| 🦶 移速 | 3 | 最高速單位。 |
| 🔊 聲音量 | 2 | 輕微移動聲，吠叫時音量=4。 |
| ⚔️ 攻擊力 | 3 | 咬合鎖定，持續傷害。 |

## 🔹 設計哲學總結

> **Shadow Protocol** 的角色平衡不依賴數值對抗，  
> 而是建立在「功能共構」的基礎上。  
>  
> 四個模組分別象徵伺服器系統中的核心機制：  
> - 滲透 → 入侵與威脅  
> - 防毒 → 掃描與清除  
> - 工作 → 維護與建構  
> - 偵測 → 感知與通報  
>  
> 當它們在戰場中互相交織，  
> 每一場戰鬥就是一場**系統防禦與滲透演算法的對抗模擬**。
